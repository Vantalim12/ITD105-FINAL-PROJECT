================================================================================
  DEPLOYMENT STEPS
================================================================================

STEP 1: COMMIT ALL FILES TO GIT
--------------------------------
Run these commands:

git status
git add app.py requirements.txt fish_conservation_data.csv best_model.pkl all_models.pkl .streamlit/
git commit -m "Prepare for Streamlit Cloud deployment"
git push origin main

STEP 2: DEPLOY ON STREAMLIT CLOUD
----------------------------------
1. Go to: https://share.streamlit.io
2. Sign in with GitHub
3. Click "New app"
4. Configure:
   - Repository: [your-username]/itd105-final-project
   - Branch: main
   - Main file path: app.py
5. Click "Deploy!"

STEP 3: WAIT FOR DEPLOYMENT
----------------------------
- Deployment takes 3-5 minutes
- Watch the logs for package installation
- Look for: "Your app is now deployed"

STEP 4: TEST YOUR APP
----------------------
1. Open the app URL: https://[your-app-name].streamlit.app/
2. Test all 5 pages:
   - SDG 14 Overview
   - Dataset Explorer
   - Model Comparison
   - Prediction Tool
   - Conclusion & Impact
3. Make a test prediction
